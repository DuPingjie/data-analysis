# data-analysis

## Requirements

In order to run the .ipynb files, you will need to install the following
Python 3 libraries :

  * ipython (6.2+)
  * numpy (1.13+)
  * pandas (0.20+)
  * matplotlib (2.1.0+)
  * scipy (0.19+)
  * scikit-learn (0.19+)
  * jupyter

## Instructions

In a terminal, type:

```bash
$ git clone https://github.com/tca19/data-analysis.git
$ cd data-analysis/
$ jupyter notebook
```

Your web browser should open. Select the corresponding td folder and complete
the .ipynb file.
